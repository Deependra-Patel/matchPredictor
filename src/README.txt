Cricket match win predictor

For running the code :
"python model.py"
this will first train the linear model, using .dat files then run on test data. One special case is also mentioned.

parse.py - parses data from data/odis folder which contains ball by ball scoring of all odi matches from 2005- 2016
classes.py - contains useful class definitions
model.py - contains code for training and testing
